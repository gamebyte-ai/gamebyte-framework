## C. v1.5 FEATURE PLAN — Öncelik Sırası

### Tier 0: Core Infrastructure (Tüm oyunlar için gerekli)

#### C1. Tween/Animation Engine
**Neden:** Her oyun animasyon kullanıyor. GSAP bağımlılığı tree-shaking bozuyor.
```typescript
import { Tween, Ease } from '@gamebyte/framework/tween';

Tween.to(sprite, { x: 200, alpha: 0, scale: 1.5 }, {
  duration: 500, ease: Ease.bounceOut, onComplete: () => {}
});

// Sequence
Tween.sequence([
  Tween.to(sprite, { y: -50 }, { duration: 200, ease: Ease.quadOut }),
  Tween.to(sprite, { y: 0 }, { duration: 300, ease: Ease.bounceOut }),
]);
```

#### C2. Particle System (2D)
**Neden:** Confetti var ama genel amaçlı particle emitter YOK.
```typescript
import { ParticleEmitter } from '@gamebyte/framework/particles';

const emitter = new ParticleEmitter({
  texture: sparkTexture,
  rate: 50, lifetime: { min: 0.5, max: 1.5 },
  speed: { min: 50, max: 150 },
  scale: { start: 1, end: 0 },
  alpha: { start: 1, end: 0 },
  color: { start: 0xFFD700, end: 0xFF4444 },
});
emitter.emit(x, y); // one-shot burst
emitter.start(); // continuous
```

#### C3. Camera System
**Neden:** Follow, zoom, bounds — her action/arcade oyunda lazım.
```typescript
import { Camera } from '@gamebyte/framework/camera';

const camera = new Camera({ viewport: { width: 540, height: 960 } });
camera.follow(player, { lerp: 0.1, offset: { x: 0, y: -100 } });
camera.setBounds(0, 0, worldWidth, worldHeight);
camera.shake(8, 300); // screenShake entegre
camera.zoom(1.5, 500); // smooth zoom
```

### Tier 1: Genre Templates (En büyük gelir fırsatları)

#### C4. Puzzle Template System
**Neden:** $8.2B pazar. Color Block Jam, Block Blast, Royal Match pattern'leri.
```typescript
import { SortingPuzzle } from '@gamebyte/framework/genre/puzzle';

const puzzle = new SortingPuzzle({
  grid: { rows: 8, cols: 8 },
  colors: 5,
  matchRule: 'adjacent-same-color', // or 'row-clear', 'group-3+'
  onMatch: (cells, color) => { /* celebration */ },
  onClear: (score) => { /* level complete */ },
});
```

Desteklenecek puzzle türleri:
- **Color/Block Sorting** (Rollic pattern)
- **Match-3+** (Royal Match pattern)
- **Tile Sliding** (15-puzzle)
- **Word Grid** (Wordle/crossword)

#### C5. Survivors/Roguelike Template
**Neden:** Habby'nin Capybara Go $100M/3 ay. Archero 2 $32M/1 ay.
```typescript
import { SurvivorsGame } from '@gamebyte/framework/genre/survivors';

const game = new SurvivorsGame({
  player: { speed: 150, health: 100, autoAttack: true, attackRange: 200 },
  waves: [...],
  upgrades: [
    { id: 'speed', name: 'Speed Boost', levels: 5, effect: (lvl) => ({ speed: 150 + lvl * 20 }) },
    { id: 'multishot', name: 'Multi Shot', levels: 3, effect: (lvl) => ({ projectiles: 1 + lvl }) },
  ],
  xpCurve: (level) => Math.floor(100 * Math.pow(1.3, level)),
});
```

#### C6. Idle/Tycoon Template
**Neden:** Growing segment, offline progress critical.
```typescript
import { IdleEngine } from '@gamebyte/framework/genre/idle';

const engine = new IdleEngine({
  resources: [{ id: 'gold', perSecond: 1 }],
  generators: [
    { id: 'miner', cost: 10, production: { gold: 1 }, costMultiplier: 1.15 },
    { id: 'factory', cost: 100, production: { gold: 10 }, costMultiplier: 1.2 },
  ],
  prestige: { threshold: 1000000, multiplier: 1.5 },
  offlineProgress: true, // Calculate earnings while app was closed
});
```

#### C8. Juice/Effects Manager
**Neden:** "Game feel" — Rollic/Voodoo kalitesi bu detaylarda.
